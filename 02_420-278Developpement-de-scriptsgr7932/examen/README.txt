Projet de Laboratoire 1

Ce projet est divisé en 5 parties, chacune implémentant des fonctionnalités spécifiques en Python.
Voici un aperçu de chaque partie (étape) et les instructions pour exécuter le code.

 Pareties du projet :

# Partie 1 : Introduction aux variables
- Fichier : ‘etape_1.py’
- Fonctionnalité : Demande à l'utilisateur de saisir son nom, son âge et la valeur de PI, puis affiche ces informations avec une précision de 8 décimales pour PI.

# Partie 2 : Structures de contrôle
- Fichier : ‘etape_2.py’
- Fonctionnalité : Génère un nombre aléatoire entre 1 et 10 et demande à l'utilisateur de le deviner. Affiche si la devinette est correcte ou non.

# Partie 3 : Listes et boucles
- Fichier : ‘etape_3.py’
- Fonctionnalité : Crée une liste de nombres de 1 à 10, puis utilise une boucle pour afficher chaque nombre additionné de 11.

# Partie 4 : Fonctions
- Fichier : ‘etape_4.py’
- Fonctionnalité : Implémente une fonction ‘somme(a, b)’ qui peut additionner deux nombres, deux chaînes de caractères, ou un nombre et une chaîne de caractères.

# Partie 5 : Manipulation de fichiers
- Fichier : ‘etape_5.py’
- Fonctionnalité : Lit un fichier texte et offre à l'utilisateur le choix de :
  1. Compter le nombre de lignes.
  2. Afficher les lignes avec un numéro de ligne pair.
  3. Compter le nombre de voyelles dans le fichier.

Merci, et bonne journée.
